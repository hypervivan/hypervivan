My Projects

Open-app sec

OpenAppSec is an open-source web application security platform designed to detect, prevent, and mitigate security threats in real time. It provides protection against common web vulnerabilities such as SQL injections, cross-site scripting (XSS), and remote code execution.

USB rubberduck

A USB Rubber Ducky is a keystroke injection tool disguised as a regular USB flash drive. When plugged into a computer, it automatically mimics a keyboard and executes pre-programmed scripts at high speed, enabling it to run commands or deliver payloads on the target system. Commonly used for penetration testing, it exploits the trust systems place in human input devices, making it a popular tool for ethical hacking and cybersecurity testing. However, it can also be misused for malicious attacks. .
